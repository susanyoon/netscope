# NetScope
NetScope is a Python-based network traffic analysis platform that processes PCAP files and extracts actionable insights about network activity.

## Features
- [ ] Parse PCAP files
- [ ] Aggregate network flows
- [ ] Traffic summaries
- [ ] Protocol statistics
- [ ] SQLite storage
- [ ] REST API
- [ ] Docker support
- [ ] Basic anomaly detection

## Tech Stack
- Python
- Scapy
- Typer
- Rich
- FastAPI
- SQLite
